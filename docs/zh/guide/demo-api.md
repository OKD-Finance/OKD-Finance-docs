# 演示 API

访问 OKD Finance 演示环境进行测试和开发。

## 演示环境

**演示 URL：** https://develop.okd.finance  
**API 基础：** https://develop.okd.finance/api  
**Swagger：** https://develop.okd.finance/swagger/gateway/

## 功能

- ✅ 完整的 API 功能
- ✅ 模拟交易
- ✅ 测试数据和账户
- ✅ Bybit 测试网集成
- ❌ 不涉及真实资金

## 开始使用

1. 在演示环境注册
2. 完成简化的 KYC
3. 获取测试代币
4. 开始开发

::: tip 提示
始终先在演示环境测试您的集成！
:::
